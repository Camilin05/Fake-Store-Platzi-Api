¿Qué es la Tienda Avila's?



Esta tienda es una aplicación de ejemplo para el curso de Django para desarrolladores web avanzado Del SENA. Esta aplicación te permite explorar muchos productos variados.

La aplicación está hecha con Django, Bootstrap y JavaScript. La API de productos está hecha con Node.js y Express.



Si tienes alguna duda o necesitas ayuda, puedes contactarnos a través de LinkedIn o Twitter.
